JSBerry is open source modular simple architecture for building Node.js applications.

This is core skeleton repository for a building new projects.

## Quick start

```bash
npm i -g jsberry@latest
```

Create new project:

```bash
  jsberry new my-project
  cd my-project
  npm start
```
